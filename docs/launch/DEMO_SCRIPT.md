# Demo Script

## Overview

This script is used for 2-minute review walkthroughs.

## Script

1. Open `/iot-maker` and announce context:
   - Independent prototype.
   - Demo-mode, operator-gated, no direct PLC control.
2. Run the 5-second reflex demo.
   - Narrate the fixed timeline to 5.0s.
3. Open logs (`/logs`) and trace the phase markers.
   - Highlight eval gate and operator gate.
4. Open query lab panel and run a source-backed query.
   - Confirm evidence and output remain bounded.
5. Open proof section in `/source` and show hash packet framing.
   - Emphasize chain target and hash-only policy.
6. Recap boundary:
   - Operator approval required.
   - no machine control path.
   - proof layer is anchored and bounded to metadata.
